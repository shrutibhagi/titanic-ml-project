# 🚢 Titanic Survival Prediction

## 📌 Overview

This project predicts whether a passenger survived using machine learning models.

## 🤖 Models Used

* Logistic Regression
* Decision Tree
* Random Forest
* SVM

## 📊 Results

Random Forest and SVM performed best.

## 🧹 Data Processing

* Handled missing values
* Converted categorical data
* Feature selection

## 📈 Evaluation

* Accuracy
* Confusion matrix
* Feature importance

## 🚀 Conclusion

This project compares multiple ML models and demonstrates how ensemble models perform better on real datasets.
